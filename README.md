# Blinkit Grocery Data Analysis using Power BI

##  Project Title
To analyze the Blinkit grocery dataset using Power BI and generate meaningful visualizations.

---

## Introduction
This project utilizes **Microsoft Power BI** to perform a comprehensive data analysis of the Blinkit grocery dataset.  
The objective is to calculate key performance indicators (KPIs) and create interactive dashboards that provide actionable insights into sales performance across different item categories and outlet types.

---

## Objectives
- Import and explore the Blinkit grocery dataset in Power BI  
- Perform data cleaning, preprocessing, and data modeling  
- Calculate key KPIs such as:
  - Total Sales
  - Average Sales
  - Number of Items
  - Average Rating
- Create interactive dashboards visualizing:
  - Sales by item type
  - Sales by outlet type and size
  - Sales by fat content
- Analyze sales performance across outlet locations and establishment years

---

##  Tools & Technologies Used
- **Dataset:** Blinkit Grocery Data (CSV)
- **Software:** Microsoft Power BI Desktop
- **Hardware Requirements:**
  - Windows system
  - Minimum 4 GB RAM

---

##  Dataset Description
The dataset contains grocery sales data across multiple outlets with the following attributes:

- **Item Fat Content:** Regular / Low Fat  
- **Item Type:** Fruits, Vegetables, Snack Foods, Soft Drinks, etc.  
- **Outlet Type:** Supermarket Type 1, 2, 3, Grocery Store  
- **Outlet Size:** Small, Medium, High  
- **Outlet Location:** Tier 1, Tier 2, Tier 3  
- **Sales & Ratings:** Item-wise sales and customer ratings  
- **Outlet Establishment Year**

---

##  Methodology / Procedure

1. Imported Blinkit grocery dataset into Power BI  
2. Performed data walkthrough to understand structure and attributes  
3. Cleaned data by handling missing values and fixing inconsistencies  
4. Applied data modeling for accurate aggregation  
5. Performed data preprocessing and transformations  
6. Created DAX measures for KPIs  
7. Designed dashboard layout  
8. Developed and formatted charts  
9. Built an interactive Power BI dashboard
